# MINIPY - 내 손안에 작은 파이썬 컴파일러

## 구현 기능
###- print()
  - 파이썬 문법 법칙에 준거하여 프린트 합니다.
  - 변수, 수, 수끼리의 연산, 문자열 출력 가능하며 쉼표(,)로 구분합니다.
###- 변수 선언
  - 문자열, 정수, 실수, 불 형태의 변수를 생성할 수 있습니다.
  - 파이썬 문법 법칙에 준거합니다
  - 이미 선언한 변수의 값을 정할 때 자료형이 변하면 오류가 발생합니다
  - input()
    - 사용자의 입력을 받아 변수에 저장합니다
    - 파이썬 문법에 준거합니다.
    - 괄호 안에 값을 입력하면 해당 값을 먼저 출력하고 같은 줄에서 입력을 받습니다. 
###- 주석 처리
  - #을 넣은 후의 문자는 무시합니다.
###- 오류 출력
  - 파이썬 문법에 준거하여 알맞지 않은 문법이 존재할 시 오류가 발생한 부분과 이유를 출력합니다.
